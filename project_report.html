BioBERT Dual Lifestyle Classification Project Report

Abstract

This pilot study demonstrates the feasibility of using fine-tuned BioBERT models for automated classification of fungal trophic modes from scientific abstracts. We achieved 86% accuracy in distinguishing fungi capable of multiple trophic modes ("dual") from those limited to single modes ("solo"), establishing a foundation for scaling to larger mycological datasets and advancing research in fungal ecology and taxonomy.

Introduction

Fungi exhibit diverse trophic strategies, with some taxa capable of occupying multiple modes (e.g., both saprotrophic and pathogenic) while others are restricted to single modes. Accurate classification of research papers on fungal trophic versatility is crucial for mycological research. This project explores automated classification using BioBERT, a biomedical language model, as a pilot experiment to assess viability for broader application.

Methodology

Data Collection

Data was sourced from curated BibTeX files containing fungal research papers, categorized into dual (fungi that can occupy multiple trophic modes) and solo (fungi limited to single trophic modes). Abstracts were extracted, cleaned of HTML/LaTeX formatting, and split 60/40 for training/testing.

Model Fine-tuning

BioBERT v1.1 (pre-trained on PubMed) was fine-tuned for binary sequence classification with learning rate 2e-5, batch size 8, and 5 epochs.

Evaluation

Performance was assessed using accuracy, precision, recall, F1-score, and confusion matrix analysis.

Conclusions

This pilot experiment successfully demonstrates BioBERT's capability for fungal trophic mode classification with 86% accuracy. The model effectively distinguishes fungi capable of multiple trophic modes from those limited to single modes, providing a scalable approach for automated literature analysis in mycology.

Limitations and Future Work

Current limitations include small dataset size and potential for domain-specific improvements. Future work involves expanding datasets, incorporating multi-modal data, and developing active learning for efficient annotation. This establishes a foundation for larger-scale mycological text mining applications.
